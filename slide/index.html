Slide

Make a presentation

(even when you got no time to make a presentation)

Also available on F-Droid and Github

Check out single-file HTML version of Slide!

Write

Content is important.

Use Takahashi method.

Create small focused slides with little text.

Nothing distracts you from writing.

Preview changes as you make them.

Emphasize

Text is scaled to fit the slide.

Add powerful `# Headlines`.

Use `*bold*` style to emphasize words.

Or show some `␣␣code();`.

Dot at the beginning of the line escapes all special syntax.

Illustrate

Images can be used as backgrounds.

Multiple images per slide can be used.

Any URL can be an image (local photo or a picture from the web).

Customize it by aligning and scaling images.

Present

Many bright themes are available

View slides directly on your device

Export slides to PDF and show it to the public
